# dec

`dec(key, ciphertext)`

<br>

Returns deciphered plaintext, as a string, of string `ciphertext` with string `key` based on Vigenère Cipher.

## Examples

```
dec(“lemon”, “lxfopvefrnhr”)
attackatdawn
```
```
dec(“cs”, “kdqngljausukkypegfv”)
ilovethisassignment
```

# enc

`enc(key, plaintext)`

<br>

Returns encrypted ciphertext, as a string, of string `plaintext` with string `key` based on Vigenère Cipher.

## Examples

```
enc(“lemon”, “attackatdawn”)
lxfopvefrnhr
```
```
enc(“cs”, “ilovethisassignment”)
kdqngljausukkypegfv
```
